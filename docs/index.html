Documentation Summary

공공직군 행정업무 슈퍼앱 문서 종합

이 문서는 기능 요구서, 기술 구조, 운영 가이드를 한눈에 볼 수 있도록 정리한 요약 화면이다. 실제 상세 내용은 각 Markdown 문서에서 확인한다.

PRD

제품 목적, 문제 정의, 주요 사용자, 핵심 기능, 우선순위를 정의한다.

  • 팀원 스케줄 관리
  • 엑셀 업무 자동화
  • 민원 대응 챗봇
  • 뉴스 기사 수집

Architecture

기술 스택, 프로젝트 구조, 모듈 역할, API 및 DB 설계를 정리한다.

  • FE: TypeScript + Vite + React
  • BE: Python + FastAPI
  • DB: SQLite

Operation

실행 방법, 자주 발생하는 에러, 기본 사용 흐름을 다룬다.

  • BE는 `uv` 가상환경 사용
  • FE는 `npm` 기반 실행
  • 로컬 SQLite 파일 운영

현재 확인된 환경

Node.js와 npm이 설치되어 있고, 백엔드는 `uv`로 `.venv`가 생성되어 있다. Python은 전역 명령이 없지만, `backend/.venv/Scripts/python.exe`를 통해 사용할 수 있다. FastAPI는 이미 설치되어 있다.
